## Testing Workflows

### Workflow 1: Create Training (as Program Officer)
1. Login as `priya@dmo.gov` / `officer123`
2. Navigate to "Create Training"
3. Fill in training details
4. Select date and time
5. System shows available halls
6. Select hall and complete form
7. Training created successfully

### Workflow 2: Nominate Participants (as Institutional Admin)
1. Login as `anjali@hospital1.gov` / `inst123`
2. Navigate to "Trainings"
3. View upcoming trainings for your institution
4. Nominate staff members
5. Nominations sent for approval

### Workflow 3: Approve Nominations (as Program Officer)
1. Login as `priya@dmo.gov` / `officer123`
2. Navigate to "Nominations"
3. Review pending nominations
4. Approve or reject with reason
5. Notifications sent to participants

### Workflow 4: Generate QR Code (as Program Officer)
1. Login as `priya@dmo.gov` / `officer123`
2. Navigate to "QR Attendance"
3. Select a training
4. Generate QR code
5. Download QR code image
6. Display at training venue

### Workflow 5: Mark Attendance (as Participant)
1. Login as `kavita@hospital1.gov` / `part123`
2. Navigate to "Scan QR"
3. Allow camera permissions
4. Scan the QR code displayed at venue
5. Attendance marked successfully

### Workflow 6: Generate Reports (any authorized role)
1. Login with appropriate account
2. Navigate to "Reports"
3. Select report type:
   - Training Report (specific training)
   - Institution Report (by institution)
   - District Summary (overall)
4. Export as PDF or CSV
5. Download report
