# How to Run Your Mobile App

## ⚠️ Requirements
1.  **Backend Server MUST be running** on your computer.
2.  **Phone and Computer MUST be on the same WiFi network**.
3.  **Android Studio** installed and set up.

## Steps to Run

1.  **Start the Backend Server (on Computer):**
    Open a terminal and run:
    ```bash
    cd server
    npm run dev
    ```
    *Keep this terminal open!*

2.  **Open in Android Studio:**
    Open a new terminal in the project folder and run:
    ```bash
    npx cap open android
    ```

3.  **Run on Device:**
    - Connect your Android phone via USB.
    - Click the green **Play** button in Android Studio.

## Troubleshooting
- **"Network Error" / "Invalid Credentials"**:
    - Check if your computer's IP address changed. Run `ipconfig` and update `src/services/api.ts`.
    - Ensure your phone is on the same WiFi as your computer.
    - Ensure the backend server is running.
- **App not updating**:
    - Run `npm run build` and `npx cap sync` after making changes.
